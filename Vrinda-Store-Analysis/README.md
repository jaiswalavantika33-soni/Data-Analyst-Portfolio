# Vrinda Store Analysis

## Project Overview
This project analyzes Vrinda Store sales data using Microsoft Excel. The dashboard provides insights into sales performance, order trends, customer demographics, and state-wise sales distribution.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## Key Insights
- Monthly sales and order trends
- Gender-wise sales analysis
- Age group analysis
- State-wise sales performance
- Order status tracking
- Channel-wise sales analysis

## Dashboard Features
- Interactive Slicers
- Dynamic Charts
- Sales KPIs
- Easy-to-understand Visualizations

## Files Included
- Vrinda Store Data.xlsx
- Dashboard Screenshots

## Conclusion
The dashboard helps identify sales trends, top-performing states, customer segments, and business performance for better decision-making.
